#  UIElements Expansions: Responsive Grid
#---------------------------------------------------------------------------

Unity UIElements Expansions is an asset series that extends Unity's UIElements.
This asset primarily adds the following features to the UnityEditor:

1.Responsive Style Sheet
Enables responsive design for inspectors and editor windows created in UIElements.
The display can be changed according to the window size.
You can use it as is, but you can also set up a detailed responsive design by editing "EditorStyleInfo".

2.Bootstrap-like Grid System
A grid system similar to the popular Web design framework called Bootstrap will be available.

Combine it with responsive stylesheets to create a modern web site-like design in an inspector or editor window
It's easy to deploy.

Here's a look at Bootstrap's grid design:.
https://getbootstrap.com/docs/4.0/layout/grid/

3.UIElements Expansions Common Features
In addition to the above, common useful tags and functions are provided.
Please see the following page for details.
https://mathru.net/unity/2202/

Unity UIElements ExpansionsはUnityのUIElementsを拡張するアセットシリーズです。
このアセットでは主に下記のような機能がUnityEditorに追加されます:

1. レスポンシブスタイルシート
UIElementsで作成されたインスペクターやエディターウインドウのデザインをレスポンシブ対応にします。
ウインドウサイズに応じた表示の切り替えが可能になります。

2. Bootstrapライクなグリッドシステム
Bootstrapという人気のWebデザインフレームワークと似たようなグリッドシステムが利用できるようになります。

レスポンシブスタイルシートと組み合わせてモダンなWebサイトのようなデザインをインスペクターやエディターウインドウに
簡単に導入することができるようになります。

Bootstrapのグリッドデザインについては以下をご覧ください。
https://getbootstrap.com/docs/4.0/layout/grid/

#  UIElements Expansions: Responsive Grid

Unity UIElements Expansions는 Unity의 UIElements를 확장하는 자산 시리즈입니다.
이 자산은 주로 다음과 같은 기능이 UnityEditor에 추가됩니다 :

1. 레스 폰 시브 스타일 시트
UIElements에서 생성 된 관리자 및 편집기 윈도우의 디자인 레스 대응합니다.
윈도우 크기에 따른 표시의 전환이 가능합니다.
그대로도 사용할 수 있지만, "EditorStyleInfo"를 편집하여 미세한 레스 폰 시브 디자인을 설정할 수 있습니다.

2. Bootstrap 좋아하는 그리드 시스템
Bootstrap이라는 인기 Web 디자인 프레임 워크와 유사한 그리드 시스템을 사용할 수 있습니다.

레스 폰 시브 스타일 시트와 함께 현대적인 Web 사이트와 같은 디자인을 관리자 및 편집기 창에
쉽게 도입 할 수 있습니다.

Bootstrap 그리드 디자인에 대해서는 다음을 참조하십시오.
https://getbootstrap.com/docs/4.0/layout/grid/

3. UIElements Expansions 공통 기능
상기 외에 공통의 편리한 태그와 기능이 제공됩니다.
